# Diagnosing Olist — A Revenue & Retention Analysis

## Overview
Analysis of 100,000+ orders across 9 relational tables from Olist's 
Brazilian e-commerce platform to diagnose a revenue plateau observed 
from mid-2018.

## Key Findings
- 75% of orders are single-item, compressing margin per delivery
- Revenue peaked in April/May 2018 and declined 14% by August 2018
- Near-zero customer retention despite replenishable top categories
- Late deliveries drop review scores by 40% (4.29 → 2.57)
- Seller count grew consistently — confirming a demand-side problem

## Tools Used
Python, Pandas, Matplotlib, Jupyter Notebook
